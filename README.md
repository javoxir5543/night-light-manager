# Night Light Manager for Linux Mint XFCE

A lightweight, stable alternative to Redshift and Generic Monitor plugin.

## Features
- No background loops – no system freezes
- Adjust warmth with Ctrl+Shift+F7/F8
- Saves last value and restores on login

## Installation
1. Clone: git clone https://github.com/your-username/night-light-manager.git
2. Run: ./install.sh
3. Set hotkeys manually in Keyboard settings

## Usage
- Ctrl+Shift+F7: Increase warmth
- Ctrl+Shift+F8: Decrease warmth

## Dependencies
- xrandr, bc, libnotify-bin (pre-installed in Mint XFCE)
